#KiiChat

[KiiCloud](https://developer.kii.com/?locale=jp)を使ったチャットのサンプルアプリです。


Android Support Library v7が必要です。  
以下のプロジェクトをワークスペースにインポートする必要があります。  

{SDK-Dir}/android-sdk-macosx/extras/android/support/v7/appcompat
